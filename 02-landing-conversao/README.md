# Landing Page Moderna com Foco em Conversao

Landing page profissional para captar clientes com visual moderno, SEO basico e formulario funcional.

## Stack

- Next.js 15 + TypeScript
- Tailwind CSS
- API Route para formulario (`/api/contact`)

## Destaques

- Hero com proposta de valor e CTA
- Animacoes suaves com CSS keyframes
- SEO basico: metadata OpenGraph + JSON-LD
- Sessao de servicos, provas sociais e FAQ
- Formulario funcional com validacao no backend
- Responsividade mobile e desktop

## Rodando localmente

```bash
npm install
npm run dev
```

## Deploy

- Publicar na Vercel
- Trocar `https://example.com` por seu dominio real em metadata e JSON-LD
